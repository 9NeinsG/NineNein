Basics

Your language will depend on what you want to accomplish.

We will break this down in 3 ways with each adding just a tad more info to understand the How.

HTML

Think of HTML as a Noun.

CSS

Think of CSS as an Adjective.

JS

Think of JS as a Verb.

HTML/CSS

Describes Noun Appearance.

HTML/JS

Gives the Noun Action.

CSS/JS

An Adverb - describing Action of the Noun.

In Reverse
Does Concept
1. HTML displays what you see on screen. Think of HTML as a Noun.
2. CSS styles how the display on screen looks. Think of CSS as an Adjective.
3. JS creates the actions on screen. Think of JS as a Verb.
4. HTML/CSS describes the Noun. Creates a Class
5. HTML/JS sets Noun into Action. Creates a Function.
6. CSS/JS designs the actions on screen. Think of CSS/JS as a Adverb.
Compound Grammar

If we Combine HTML/CSS we are describing(Adjective) what is shown(Noun).
If we Combine HTML/JS we are setting what is shown(Noun) into action(Verb).
If we Combine CSS/JS we are describing(Adjective) an Action(Verb) creating an Adverb.

Write A Sentence

Put it all together and a full sentence is written - or a Line of Code.
Once we have a few of these Lines of Code(sentences) we then have a Script aka book, movie, dialogue, etc.
