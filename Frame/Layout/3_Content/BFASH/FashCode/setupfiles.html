Here I will provide a simple file to save alot of time and for an instant setup of these packages; as well as how to install them.

Open Sublime Text
1. Select [Preferences] in the NavBar - Select [Browse Packages]
2. _Go Back one Folder and Drag Rar file into that folder
3. We going to merge this folder with the Download.
4. Drag Download "packages" into folder and Merge.
*Delete or Save the Rar File(A backup is always good)
5. Open Packages.
6-7. Finally go to [Packages>User>SublimeLinter.sublime.settings]
8. Double check the path - in most cases just replace *Username* with your ComputerName.
9.Done!

**If you can not find "AppData" make sure to change your computer settings to Show Hidden Files.
